# Dictee

A simple minimalist text editor written in C heavily inspired by kilo

## Build

```bash
make
```

## Usage

```bash
./dictee
```

## Debug

```bash
gdb --args dictee test.txt
b 165
run
print somevariable
c continue
s step
```
